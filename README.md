**2. 사용 기술**
- OS : Windows10 64bit
- SERVER : Apache Tomcat v9.0
- TOOL : Java JDK-1.7
- Framework : Spring Framework 5.0 Mybatis 3.2.2
- DBMS : Oracle 11g Release2
- Language : Java, HTML, CSS, JavaScript, JSP, jQuery, BootStrap, AJAX

**3. 개발기간 & 참여인원**

- 구매한 상품 상품 리뷰 작성 및 표시
